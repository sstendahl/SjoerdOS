# SjoerdOS

Personal project to run my own UBlue Spin. 

This will be mainly there for my personal use, so any choices made are there to reflect my own workflow and needs. 
This entire README is a work in progress, as well as the goals of this image to the image name :) 

But my main philosophy is that it should match upstream Fedora and GNOME as close as reasonable. Specific tweaks, or change of system libraries will be avoided unless deemed absolutely neccesary.

Support is not gauranteed, but feel free to run this spin, I will reply to issues or PR nonetheless.
